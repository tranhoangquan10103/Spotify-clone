## 📖 Giới thiệu

Dự án **Spotify Clone** này là một ứng dụng web mô phỏng lại trải nghiệm nghe nhạc của Spotify. Dự án được phát triển nhằm mục đích rèn luyện tư duy lập trình, kỹ năng xây dựng UI/UX phức tạp và làm việc với các API âm nhạc.

*Lưu ý: Đây là dự án học tập phi thương mại, mọi bản quyền âm thanh và hình ảnh đều thuộc về tác giả gốc và Spotify.*

## ✨ Tính năng nổi bật

- **🎧 Trình phát nhạc:** Phát, tạm dừng, chuyển bài tiếp theo/trước đó.
- **📚 Quản lý danh sách phát:** Tạo, chỉnh sửa và xóa playlist cá nhân.
- **🔍 Tìm kiếm:** Tìm kiếm bài hát, nghệ sĩ và album nhanh chóng.
- **📱 Responsive Design:** Giao diện hiển thị tốt trên cả Desktop, Tablet và Mobile.
- **🔐 Xác thực người dùng:** Đăng nhập, đăng ký tài khoản (JWT/OAuth).

## 🚀 Công nghệ sử dụng

Dự án được xây dựng với các công nghệ hiện đại:

### Frontend
- **Framework:** [React.js](https://reactjs.org/) / [Next.js](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) hoặc SCSS
- **State Management:** Redux Toolkit / React Context

### Backend & Database (Nếu có)
- **Runtime:** [Node.js](https://nodejs.org/)
- **Framework:** Express.js
- **Database:** MongoDB / PostgreSQL

### API & Tools
- **Music API:** Spotify Web API (hoặc API giả lập tự build)
- **Deploy:** Vercel (Frontend), Render (Backend)
